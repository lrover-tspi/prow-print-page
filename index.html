PROW Feature Report Generator

This page generates printable PDF reports for Public Right-of-Way (PROW) accessibility assessment features.

Usage

Access the report page with URL parameters:

print_feature.html?service=<feature_service_url>&layer=sidewalks&oid=123

Required Parameters

  • oid - The OBJECTID of the feature to display

Optional Parameters

  • service - Feature service URL (defaults to NWMCC service)
  • layer - Layer type: sidewalks, crosswalks, curbramps, barriers (default: sidewalks)
  • layerIndex - Override the layer index in the feature service
  • idField - Override the ID field name (e.g., segment_id, c_path_id)
  • title - Custom report subtitle
  • autoprint - Set to "true" to auto-open print dialog

Example

print_feature.html?service=https://services.arcgis.com/.../FeatureServer&layer=sidewalks&oid=456&title=City%20of%20Example&autoprint=true

Test with default service (OBJECTID 1)
